# frame-design

Skill para crear y editar assets SVG con frame-svg usando todo el potencial del framework.

</div>

## Uso

Se activa automáticamente cuando el usuario pide ayuda con un diseño en frame-svg: crear un asset SVG, diseñar una card, banner, diagrama o dashboard, editar un `.frame` existente, o cualquier tarea relacionada con componentes frame-svg.

## Flujo de trabajo

```
Determinar contexto → Cargar references → Diseñar e implementar → Self-check
```

## References

| Fichero | Contenido |
|---------|-----------|
| `file-model.md` | Estructura del proyecto, Page como raíz, imports con `@/` |
| `theme.md` | Variables `$*`, dark/light mode, cuándo usar raw colors |
| `layout.md` | Stack, Grid, Box, Spacer — composición y alineación |
| `primitives.md` | Page, Stack, Box, Grid, Spacer, Text, Circle, Line, Icon |
| `compound.md` | Card, Avatar, Callout, FeatureList, FileTree, KeyCombo, Stat |
